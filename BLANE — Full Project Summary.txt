SQL Tables (run in Supabase SQL Editor)
profiles table
sqlcreate table profiles (
  id                   uuid primary key references auth.users(id) on delete cascade,
  full_name            text,
  age                  int,
  sex                  text,
  height_cm            numeric,
  weight_kg            numeric,
  activity_level       text,
  goal                 text,
  dietary_restrictions text[],
  medical_conditions   text[],
  updated_at           timestamptz
);

alter table profiles enable row level security;

create policy "own profile" on profiles
  for all using (auth.uid() = id);


health_logs table
sqlcreate table health_logs (
  id               uuid    default gen_random_uuid() primary key,
  user_id          uuid    references auth.users(id) on delete cascade,
  logged_at        date    not null default current_date,
  weight_kg        numeric,
  sleep_hours      numeric,
  water_ml         integer,
  calories_burned  integer,
  updated_at       timestamptz default now(),
  unique (user_id, logged_at)
);

alter table health_logs enable row level security;

create policy "own logs" on health_logs
  for all using (auth.uid() = user_id);

Module 01 — Body Feedback Loop Details

Tracks: weight (kg), sleep (hrs), water (cups × 300ml), calories burned
On save: upserts health_logs, updates profiles.weight_kg if changed
Recalculates: BMI, TDEE, calorie target, macros (Mifflin-St Jeor formula)
Drift alerts: weight spike/drop, sleep deficit (<6hrs), low hydration (<1800ml), low activity (<75% TDEE)
7-day sparklines: SVG mini charts for weight, sleep, activity

Module 02 — Health Drift Detection Details

Algorithm: linear regression slope on 7-day health_logs data
Metrics: weight trend, sleep pattern (+ irregularity via CV%), hydration, BMI
Scoring: weighted health score 0–100 (Weight 30%, BMI 25%, Sleep 25%, Hydration 20%)
Score labels: Excellent (85+), Good (70+), Fair (55+), Needs Attention (40+), At Risk (<40)
Charts: SVG line charts per metric with gradient fill + regression trendline + endpoint dot
Re-runs automatically every time Module 01 logs new data

Formulas Used
BMR (Mifflin-St Jeor):

Male: 10w + 6.25h - 5age + 5
Female: 10w + 6.25h - 5age - 161

TDEE: BMR × activity multiplier
Goal-adjusted calorie target:

Lose weight: TDEE × 0.85
Gain muscle: TDEE × 1.10
All others: TDEE × 1.00

Macro split: Protein 30% (÷4 kcal/g) · Carbs 45% (÷4) · Fats 25% (÷9)
BMI: weight_kg ÷ (height_m)²
