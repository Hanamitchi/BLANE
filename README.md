 BLANE - Adaptive Nutrition Recommendation System Integrating Real-Time Health Data and Local Food Availability

 This project is a capstone study entitled "An Adaptive Nutrition Recommendation 
 System Integrating Real-Time Health Data and Local Food Availability". 
 The system is designed to provide intelligent, personalized, and context-aware meal 
 recommendations based on user health data, dietary preferences, budget, and locally available food resources.
